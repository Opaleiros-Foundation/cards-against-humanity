FAQ

Perguntas Frequentes

P: Por que não consigo fazer login (carregamento infinito)?

R: Atualmente estamos utilizando serviços gratuitos para manter o servidor, então em certos horários de pico o servidor pode ficar sobrecarregado. Tente novamente ou em outro dia.

P: Posso criar minhas próprias cartas?

R: No momento apenas é possível jogar com as cartas disponíveis, caso deseje adicionar uma carta, acesse o link: Adicionar Carta.

P: Como convidar amigos?

R: Quando criar a sala, um código vai estar disponível para compartilhar com seus amigos. Eles devem inserir o código na tela inicial. Eles também podem procurar a sala em "Encontrar Partida".

P: Como faço para entrar em contato com o suporte?

R: Se você tiver algum problema ou dúvida, nossa equipe de suporte está pronta para ajudar! Basta acessar a seção de "Suporte" no site ou enviar um e-mail para silvanosilvino@hotmail.com.

P: Onde posso jogar este jogo?

R: Em navegadores, tanto desktop quanto mobile.

P: O jogo é adequado para todas as idades?

R: Nosso jogo é voltado para um público adulto e pode conter humor irreverente ou linguagem que não é adequada para crianças. Recomendamos que você jogue com um grupo de amigos que compartilham um senso de humor similar.

P: O que é este jogo?

R: Nosso jogo é inspirado no famoso "Cartas Contra a Humanidade", onde os jogadores devem combinar perguntas e respostas de forma criativa para formar frases engraçadas ou absurdas. O objetivo é se divertir com os amigos e criar momentos de humor.
